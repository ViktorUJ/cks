# 08

Cilium Endpoint (or cep) resource can be used to retrieve a list of all endpoints managed by Cilium

To check all cilium endpoints we might use the next command:

```sh
kubectl get cep --all-namespaces
```

We have to create the next policy to allow access to **/public** path:

```yaml
# vim public.yaml

apiVersion: "cilium.io/v2"
kind: CiliumNetworkPolicy
metadata:
  name: "public"
  namespace: "production"
spec:
  endpointSelector:
    matchLabels:
      app: app
  ingress:
  - fromEntities:
     - cluster
    toPorts:
    - ports:
      - port: '80'
        protocol: TCP
      rules:
        http:
          - path: "/public.*"
```

```sh
k apply -f public.yaml
```

We have to create the next Cilium policy to allow access form  **finance** namespace to **/private** path:

```yaml
# vim private.yaml
---
apiVersion: "cilium.io/v2"
kind: CiliumNetworkPolicy
metadata:
  name: "private"
  namespace: "production"
spec:
  endpointSelector:
    matchLabels:
      app: app
  ingress:
  - fromEndpoints:
    - matchLabels:
        k8s:io.kubernetes.pod.namespace: finance
    toPorts:
    - ports:
      - port: '80'
        protocol: TCP
      rules:
        http:
          - path: "/private.*"
...
```

```sh
k apply -f private.yaml
```
