## Задание 1. Под `nginx1233` с livenessProbe (exec)

```bash
kubectl create namespace web-ns
```

```yaml
apiVersion: v1
kind: Pod
metadata: {name: nginx1233, namespace: web-ns}
spec:
  containers:
  - name: nginx1233
    image: nginx
    livenessProbe:
      exec:
        command: ["ls", "/var/www/html/"]
      initialDelaySeconds: 10
      periodSeconds: 60
```

## Задание 4. Починить устаревший манифест и задеплоить

Файл `/var/work/109/app-old.yaml` использует удалённую `apps/v1beta1` и не содержит
обязательного `selector`. Исправляем на актуальную версию:

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: app-21
spec:
  replicas: 1
  selector:
    matchLabels: {app: app-21}
  template:
    metadata:
      labels: {app: app-21}
    spec:
      containers:
      - name: app-21
        image: viktoruj/ping_pong:alpine
```

```bash
kubectl apply -f /var/work/109/app-old.yaml
kubectl get deploy app-21
```

Актуальную версию API можно подсмотреть: `kubectl explain deployment` и
`kubectl api-versions`.
