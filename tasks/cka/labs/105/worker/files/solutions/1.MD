# Лаба 105 — эталонное решение (ConfigMap, Secret, env)

## Задание 1. Secret + под с env из секрета (неймспейс `dev-db`)

```bash
kubectl create namespace dev-db
kubectl create secret generic dbpassword -n dev-db --from-literal=pwd=my-secret-pwd
```

```yaml
apiVersion: v1
kind: Pod
metadata: {name: db-pod, namespace: dev-db}
spec:
  containers:
  - name: db-pod
    image: mysql:8.0
    env:
    - name: MYSQL_ROOT_PASSWORD
      valueFrom:
        secretKeyRef: {name: dbpassword, key: pwd}
```

## Задание 2. ConfigMap → переменная окружения (неймспейс `app-cfg`)

```bash
kubectl create namespace app-cfg
kubectl create configmap app-config -n app-cfg --from-literal=COLOR=blue
```

```yaml
apiVersion: v1
kind: Pod
metadata: {name: cfg-pod, namespace: app-cfg}
spec:
  containers:
  - name: cfg-pod
    image: nginx
    envFrom:
    - configMapRef: {name: app-config}
```

## Задание 3. ConfigMap из файла → том в деплое (неймспейс `app-z`, `/appConfig`)

```bash
kubectl create namespace app-z
kubectl create configmap config -n app-z --from-file=/var/work/105/config.yaml
```

```yaml
apiVersion: apps/v1
kind: Deployment
metadata: {name: app-z, namespace: app-z}
spec:
  replicas: 1
  selector: {matchLabels: {app: app-z}}
  template:
    metadata: {labels: {app: app-z}}
    spec:
      containers:
      - name: app-z
        image: viktoruj/ping_pong:alpine
        volumeMounts:
        - {name: config, mountPath: /appConfig}
      volumes:
      - name: config
        configMap: {name: config}
```

## Задание 4. Секрет + ConfigMap, смонтированные в под (неймспейс `prod-apps`)

```bash
kubectl create namespace prod-apps
kubectl create secret generic prod-secret -n prod-apps --from-literal=var1=aaa --from-literal=var2=bbb
kubectl create configmap prod-config -n prod-apps --from-literal=config.yaml="test config"
```

```yaml
apiVersion: v1
kind: Pod
metadata: {name: prod-app, namespace: prod-apps}
spec:
  containers:
  - name: app1
    image: viktoruj/cks-lab:latest
    command: ["sleep","60000"]
    volumeMounts:
    - {name: config, mountPath: /app/configs}
  - name: app2
    image: viktoruj/cks-lab:latest
    command: ["sleep","60000"]
    volumeMounts:
    - {name: secret, mountPath: /app/secrets}
  volumes:
  - name: config
    configMap: {name: prod-config}
  - name: secret
    secret: {secretName: prod-secret}
```
