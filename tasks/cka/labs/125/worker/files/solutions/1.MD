# Лаба 125 — эталонное решение (DNS и CoreDNS)

Ускорители:

```bash
alias k=kubectl
export do="--dry-run=client -o yaml"
```

## 1. namespace + deployment

```bash
kubectl create namespace dns
kubectl -n dns create deployment web --image=viktoruj/ping_pong:latest --replicas=2
```

## 2. ClusterIP web-svc (обычная A-запись → один ClusterIP)

```bash
kubectl -n dns expose deployment web --name=web-svc --port=80 --target-port=8080
kubectl -n dns get svc web-svc            # запомнить ClusterIP (пригодится в задании 6)
```

## 3. Headless web-h (A-записи → IP всех Pods)

```bash
kubectl -n dns create service clusterip web-h --clusterip="None" --tcp=80:8080
kubectl -n dns patch svc web-h -p '{"spec":{"selector":{"app":"web"}}}'
kubectl -n dns get endpoints web-h        # должны появиться IP Pods web
```

Проверка DNS (по желанию, из временного Pod):

```bash
kubectl -n dns run tmp --rm -it --image=busybox --restart=Never -- \
  sh -c 'nslookup web-svc; echo ---; nslookup web-h'
# web-svc → один ClusterIP; web-h → список IP всех Pods
```

## 4. Pod client с dnsConfig ndots=2

```bash
cat <<'EOF' | kubectl apply -f -
apiVersion: v1
kind: Pod
metadata:
  name: client
  namespace: dns
spec:
  dnsPolicy: ClusterFirst
  dnsConfig:
    options:
    - name: ndots
      value: "2"
  containers:
  - name: client
    image: viktoruj/ping_pong:alpine
EOF
```

Проверить DNS прямо из этого Pod (в теге `alpine` есть `nslookup`/`wget`):

```bash
kubectl -n dns exec client -- nslookup web-svc
kubectl -n dns exec client -- nslookup web-h
```

## 5. CoreDNS: forward домена mycorp.local на 10.10.0.10

Правим ConfigMap `coredns` и добавляем stub-блок для домена:

```bash
kubectl -n kube-system edit configmap coredns
```

Добавить в Corefile отдельный блок (рядом с основным `.:53 { ... }`):

```
mycorp.local:53 {
    errors
    cache 30
    forward . 10.10.0.10
}
```

Применить изменения:

```bash
kubectl -n kube-system rollout restart deployment coredns
```

## 6. JSONPath: ClusterIP web-svc в файл

```bash
mkdir -p /home/ubuntu/answers
kubectl -n dns get svc web-svc -o jsonpath='{.spec.clusterIP}' \
  > /home/ubuntu/answers/clusterip.txt
```
