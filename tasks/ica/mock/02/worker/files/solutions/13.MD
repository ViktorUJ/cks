# Task 13: Install Istio with Demo Profile

```bash
kubectl config use-context cluster3-admin@cluster3

istioctl install -f - -y <<'EOF'
apiVersion: install.istio.io/v1alpha1
kind: IstioOperator
spec:
  profile: demo
  hub: docker.io/istio
  tag: 1.26.3
  revision: 1-26-3
  components:
    egressGateways:
    - name: istio-egress-gateway
      enabled: true
    pilot:
      k8s:
        resources:
          requests:
            cpu: 100m
            memory: 200Mi
EOF

# Verify installation
kubectl get deployment -n istio-system
istioctl version --revision 1-26-3
```
